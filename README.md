# noe-sounds

a collection of sounds from hardware synths such as the tempest, machinedrum, and monomachine by noeisgirl to be used on strudel

to use this inside of strudel, enter the following line in your strudel pattern:

```
samples('github:kyrsive/noe-sounds')
```
